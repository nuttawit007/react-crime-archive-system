# Thailand Crime Archive System

Interactive web platform for exploring crime cases across Thailand through maps, provinces, and case categories.

This project was developed as part of the Client Side Web Development course using React + Vite.

## 📚 Project Overview

Thailand Crime Archive System is a web application designed to allow users to explore crime-related information across Thailand through an interactive interface.

The system allows users to:
- Explore crime cases by province
- View crime categories
- Discover cases using an interactive Thailand map
- View detailed information about each case
- Get recommended cases through a quiz system

The main goal of the project is to demonstrate Client Side Web Development concepts using React while integrating data visualization with geographic interaction.

## 🚀 Installation

Clone Project:
```
git clone https://github.com/nuttawit007/react-crime-archive-system.git
cd react-cst-project
```

install dependency:
```
npm install
```

Run Development Server:
```
npm run dev
```

The application will run at:
```
http://localhost:5173
```

## ⚠️ Important Note

All data used in this project are mock data created for educational purposes only.

This includes:
- crime cases
- province data
- number of cases
- categories
- statistics

The system does not represent real crime records and should not be interpreted as real-world data.
